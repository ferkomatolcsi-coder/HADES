# 💀 HADES
### Hard Disk Explorer & Storage

> *「土曜日の午後。一人の俳優。一台のノートパソコン。他にやることは何もなかった。」*

---

## これは何？

HADESは、外付けハードドライブに何が入っているかを正確に把握するディスクインデックスシステムです — たぶんあなた自身よりも詳しく。

スキャンし、バージョンを追跡し、変更を検出し、すべてを美しいExcelファイルに吐き出します。やらない理由がないでしょう。

---

## 何ができるの？

- 🔍 **自動ディスク検出** – 繋ぐだけ、HADESが気づく
- 📁 **ファイルインデックス** – すべてのファイル、すべてのディスク、一箇所に
- 🔄 **変更追跡** – 何があったか、今何があるか、何が消えたか、全部わかる
- 📊 **Excelエクスポート** – ダッシュボード + ディスクごとの個別シート
- ⚡ **スマートキャッシュ** – 変更がなければ再生成なし（29万6千ファイルを読み直すのはスポーツじゃない）
- 💾 **SQLiteデータベース** – 永続的、堅牢、絶対に忘れない

---

## インストール

```bash
git clone https://github.com/ferkomatolcsi-coder/HADES.git
cd HADES
pip install -r requirements.txt
```

---

## 使い方

**スキャン** – 接続されたディスクをインデックス化:
```bash
python3 hades_scan.py
```

**エクスポート** – Excelファイルを生成:
```bash
python3 hades_export.py
```

結果: HADESフォルダに `HADES_export_日付_時刻.xlsx`

---

## 動作環境

- macOS または Linux
